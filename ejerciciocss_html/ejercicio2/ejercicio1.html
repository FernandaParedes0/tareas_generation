Este es mi primer documento

¿Que contiene?

  • Ejercicio #1 Crea una página HTML usando 5 etiquetas diferentes y coloca algún contenido dentro.
  • Ejercicio #2 Cree una página HTML que muestre una imagen. Ubicarla dentro de un div.
  • Ejercicio #3 Crea una entrada de blog sobre la ciudad de la que eres. Debe tener un título, algo de texto, una lista, enlaces e imágenes. Ten cuenta de organizar el contenido de etiquetas para órdenes el contenido y que sea a futuro más sencillo aplicar estilos.
  • Ejercicio #4 Crea un anuncio sobre un producto que te guste. Puedes tomar una imagen de referencia de internet, agregar textos, títulos y elementos adicionales, todos estos encapsulados en las etiquetas correspondientes.
  • Ejercicio #5 Crea una página personal sobre ti. Pueden usar como base el ejercicio del cv en HTML.

¿Por qué debo hacer mi tarea

Para profundizar y reforzar el conocimiento de la clase, mejorar la comprensión de la materia y es una puerta de entrada para practicar lo aprendido
